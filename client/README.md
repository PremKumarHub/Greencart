# Greencart Frontend

This is the frontend portion of the Greencart e-commerce platform, built with React and Vite.

For full project documentation, including setup and running instructions, please see the [main README](../README.md) in the root directory.

## Available Scripts

- `npm run dev`: Runs the app in development mode.
- `npm run build`: Builds the app for production.
- `npm run lint`: Runs ESLint for code quality checks.
